Getting Started with Vim

Basic Commands in Vim

  1. To open a file: 
    vim filename.txt
  2. To save and quit: 
    :wq
    Press Esc to switch back to Normal mode before typing this command.
  3. To quit without saving: 
    :q!
    This is useful if you’ve made changes you don’t want to keep.

Vim Modes

  • Normal mode: The default mode for navigating and editing commands
  • Insert mode: Enter text by pressing i to go into Insert mode, allowing you to type freely
  • Visual mode: Used for selecting and highlighting text

Common Commands

  • dd - Delete the current line
  • yy - Copy (yank) the current line
  • p - Paste the copied text below the current line
  • u - Undo the last action
  • Ctrl + r - Redo the last undone action

Sample Output in Vim

When you type :w to save, Vim will display the following confirmation:

"filename.txt" written
Press ENTER or type the command to continue
